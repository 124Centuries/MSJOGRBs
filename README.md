# MSJOGRBs
A simple toolkit for simulation of multi-satellite joint obervation of gamma-ray bursts.(MSJOGRBs)

We already give a exampe in this zip file, and you can run 'lc_setting.py' to see our example spectrum and profile.
More detailed information can see the 'Introduction' in the zip file.

We won't supply true RSP files here. For users to experience the code here, we already supply the matrix data, ebounds data and so on. In other words, there is no need to generate matrix again if you only want to have an try.
